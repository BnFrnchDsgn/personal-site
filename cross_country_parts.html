Summary
Working in a project team, I was assigned design and development lead for the new Cross Country Parts website. I delivered a new, modern web design and built out the entire website as part of a larger campaign to improve the Cross Country Parts digital presence & brand awareness.
What I Did
There were a variety of areas in this project that I was directly involved in, which included:
  • New site-map and wire-frames were planned to map out the new website structure.
  • New web design, which better reflected the brand goals & values of the business.
  • Development of the new website using best-practices front-end development techniques.
Reflection
This was the largest app project that I had worked at this time, and I took this opportunity to ensure every design decision and recommendation was the best it could be. Focusing on even the smallest assumptions and details helped produce a final design that surpassed the client's expectations.
Programs Used
There were quite a few different deliverables, requiring me to be proficient in a variety of different tools. These included:
  • Google Workspace for handling project notes and communication
  • Adobe Illustrator to create graphic design assets used in the final web design
  • Adobe XD when designing out the new Cross Country Parts web design
  • Visual Studio Code as the preferred IDE when developing the site
  • ShoutCMS was the web CMS platform used to develop the website on
